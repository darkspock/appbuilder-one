# Privacy Policy — AppBuilder One

**Last updated:** March 27, 2026

## Summary

AppBuilder One does not collect, store, or transmit any user data. It is a Claude Code plugin that runs entirely on your local machine.

## What This Plugin Does

AppBuilder One provides skills (slash commands) that help you build web applications. All processing happens locally within your Claude Code session.

## Data Collection

**We collect no data.** Specifically:

- No analytics or telemetry
- No usage tracking
- No personal information
- No cookies
- No network requests to external servers
- No data shared with third parties

## Data Processing

All data processed by this plugin (code, designs, configurations) stays within your local Claude Code session and your project files. The plugin does not send data anywhere.

## Third-Party Services

The plugin may recommend third-party services (Supabase, Vercel, AWS, etc.) during setup. If you choose to use those services, their respective privacy policies apply. AppBuilder One does not share your data with them.
